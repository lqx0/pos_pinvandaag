What is POS Pin Vandaag?

POS Pin Vandaag is an Odoo module designed to provide businesses and organizations with a free and easy to use solution for controlling their payment terminals

Wich kind of terminals can POS Pin Vandaag manage?

POS Pin Vandaag can manage the following terminals:

  • Worldline
  • CCV

How can I sing up for the module?

You can sign up for the module by sending an email to helpdesk@pinvandaag.nl and we will contact you as soon as possible.
